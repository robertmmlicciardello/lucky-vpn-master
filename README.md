## Technology Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express
- **Database**: MySQL with Sequelize ORM
- **Mobile**: Capacitor for Android
- **Authentication**: JWT tokens
